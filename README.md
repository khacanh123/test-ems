## Installation

Use the package manager [Yarn](https://yarnpkg.com/getting-started/install) to install.

```bash
npm i yarn --global
```

## Usage

```
yarn install

yarn dev
```

## Project structure

```
App.tsx > route > layout > pages > component
```

# Build Project: `yarn build`

- node-version: [14.x]
